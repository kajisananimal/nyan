# ビックリマン風キャラクター診断（静的Webサイト）

## 使い方（ローカル）
- フォルダを展開して `index.html` を開くだけで動きます（保存なし / ローカル判定）。

## デプロイ（簡単）
### GitHub Pages
1. このフォルダの中身をリポジトリ直下へアップ
2. Settings → Pages → Deploy from a branch（main / root）を選択

### Netlify
- Siteを作成 → このフォルダをドラッグ＆ドロップでアップロード

## URL仕様
- 結果: result.html?type=moon_cat など
- 相性: compatibility.html?me=moon_cat&you=sun_dog

画像は `assets/` にあります。


【スマホから開けるリンクについて】
このサイトは静的サイトなので、Netlify / GitHub Pages に公開するとURLが発行され、スマホからそのURLで開けます。
